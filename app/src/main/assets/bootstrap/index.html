Android開發

Android早期由“Android之父”之稱的Andy Rubin創辦，Google於2005年並購了成立僅22個月的高科技企業Android。

IOS開發

蘋果公司最早於2007年1月9日的Macworld大會上公布這個系統，最初是設計給iPhone使用的， 後來陸續套用到iPod touch、iPad以及Apple TV等產品上。iOS與蘋果的Mac OS X操作系統壹樣， 屬於類Unix的商業操作系統。原本這個系統名為iPhone OS

嵌入式底層開發

嵌入式系統是軟件和硬件的綜合體，還可以涵蓋機械等附屬裝置。國內普遍認同的嵌入式系統定義為： 以應用為中心，以計算機技術為基礎，軟硬件可裁剪，適應應用系統對功能、可靠性、成本、體積、 功耗等嚴格要求的專用計算機系統。

Bootstrap框架

Bootstrap是美國Twitter公司的設計師Mark Otto和Jacob Thornton合作基於HTML、CSS、JavaScript 開發的簡潔、直觀、 強悍的前端開發框架，使得 Web 開發更加快捷。

你的網站和應用能在bootstrap的幫助下通過同一份代碼快捷、有效的適配手機、平板、PC設備
